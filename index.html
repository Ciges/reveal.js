Pasos a seguir

  • Descargar el paquete OVA con la máquina virtual desde https://miservidor.com/files/Aula_Tegnix.ova
  • Instalar VirtualBox (u otro) desde www.virtualbox.org
  • Importar el servicio virtualizado
  • Arrancar la máquina e iniciar sesión con el usuario estudiante y clave estudiante

Notas para el uso de la máquina virtualizada

  • Con la teclas CTRL derecha + f se activa/desactiva la pantalla completa
  • Con el comando sudo se ejecutan órdenes como root

Convertirnos en root

sudo su -

Abrir una nueva terminal

sudo gnome-terminal

Fin de la sesión :-)
